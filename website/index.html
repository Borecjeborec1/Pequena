Pequena

Lightweight cross-platform desktop app framework designed for simplicity and speed

  • Easy to use

    Easy and stable keywords, helps you learn really fast.

  • Lightweight

    Bundled core exe application is around 8MB.

  • Cross Platform

    Doesn't matter if the target is Linux or PiOS.

  • Open-Source

    The source code is available on Github.

  • The core framework app is just 8MB in size, making it easy to distribute and install.
  • Pequena supports Node.js built-in modules, and if that's not enough, python will do the job.
  • Pequena apps use minimal memory, making them fast and efficient even on older computers.
  • Pequena's API is designed to be simple and intuitive, making it easy to create desktop apps quickly.
